>우리동네 대여서비스 :: Client Server  
>가까웃 동네 이웃들끼리 필요한 물건을 대여, 나눔할 수 있도록 도와주는 매칭 서비스
>#RestAPI사용

## 주요 기능
* **회원관리**  
    * **일반회원CRUD, 이메일 인증, kakao우편번호 서비스, kakao회원 CRUD, kakao회원 메세지보내기**    
* 요청게시판  
* 나눔게시판  
* 동네생활 게시판  
* QnA  
* 채팅
